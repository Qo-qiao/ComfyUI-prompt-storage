# ComfyUI-prompt-storage

一个用于管理和存储图片提示词信息的 ComfyUI 插件。

**[📃English](./README.md)**

## 项目简介

ComfyUI-prompt-storage 是一款专门的 ComfyUI 插件，用于管理和存储图片的提示词信息。它允许用户从输出目录加载图像文件，查看其提示词详情，并输出提示词信息以供其他节点使用。

### 核心优势
- **简单直观**：使用方便，界面清晰，支持中英文双语显示
- **图像文件支持**：从输出目录加载图片，支持多种格式
- **提示词管理**：显示和管理图像文件的提示词信息，支持正向/负向提示词多选
- **标签系统**：支持为图片添加标签，便于分类和筛选
- **预设模板**：支持保存和管理提示词模板，快速复用
- **无缝集成**：输出提示词供其他 ComfyUI 节点使用
- **视觉界面**：提供图像文件的视觉预览，支持卡片视图和列表视图

## 核心功能

### 提示词存储器节点
- **图像文件加载**：从输出目录加载图像文件
- **提示词显示**：显示与图像文件关联的正向和负向提示词
- **提示词选择**：支持多选正向提示词，单选负向提示词
- **视觉预览**：显示图像文件的缩略图，支持左右切换浏览
- **悬停预览**：鼠标悬停显示详细信息预览
- **编辑功能**：支持编辑文件名、标签、风格、提示词等信息
- **标签筛选**：按标签筛选图片
- **标签提示**：多个标签请使用英文逗号`,`分隔，例如：`人像,写实,风景`（请勿使用中文逗号`，`）

### 预设模板存储器节点
- **模板管理**：保存、编辑、删除提示词模板
- **模板分类**：按风格分类管理模板
- **快速应用**：一键应用模板到工作流
- **搜索功能**：快速搜索模板
- **文件格式**：支持读取 TXT、JSON、MD 格式的模板文件

## 安装说明

### 1. 基本安装

1. **克隆或下载插件**：
   - 将插件文件夹放入 `ComfyUI/custom_nodes/` 目录
   - 文件夹名称应为 `ComfyUI-prompt-storage`

2. **安装依赖**：
   - 插件除了 ComfyUI 的默认要求外，没有其他额外依赖

### 2. 使用方法

1. **将节点添加到工作流**：
   - 在 ComfyUI 中，从 "prompt" 类别添加 "提示词存储器(Image Prompt Storage)" 或 "预设模板存储器(Prompt Template)" 节点

2. **配置提示词存储器节点**：
   - 设置文件路径为输出目录（默认为 ComfyUI 的输出目录）
   - 从下拉列表中选择一个媒体文件
   - 点击"查看"按钮打开视觉界面

3. **配置预设模板存储器节点**：
   - 点击"管理模板"按钮打开模板管理界面
   - 创建、编辑或删除模板

4. **使用输出**：
   - 节点输出正向和负向提示词
   - 将这些输出连接到需要提示词输入的其他节点

## 工作流示例

### 基本使用

1. 将 "提示词存储器(Image Prompt Storage)" 节点添加到工作流
2. 从下拉列表中选择一个媒体文件
3. 点击"查看"按钮打开视觉界面
4. 在界面中选择需要的正向/负向提示词
5. 将输出连接到其他节点
6. 执行工作流以使用提示词

### 使用预设模板

1. 将 "预设模板存储器(Prompt Template)" 节点添加到工作流
2. 点击"管理模板"创建或选择模板
3. 将模板输出连接到其他节点
4. 执行工作流
