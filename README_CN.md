# ComfyUI-prompt-storage

一个用于管理和存储图片提示词信息的 ComfyUI 插件。

**[📃English](./README.md)**

## 项目简介

ComfyUI-prompt-storage 是一款功能丰富的 ComfyUI 插件，用于管理和存储图片提示词信息，同时支持文本内容的修改和处理。它提供了多个专业节点，帮助用户高效管理提示词、模板和进行文本操作。

### 核心优势
- **简单直观**：使用方便，界面清晰，支持中英文双语显示
- **图像文件支持**：从输出目录加载图片，支持多种格式
- **提示词管理**：显示和管理图像文件的提示词信息，支持正向/负向提示词多选
- **标签系统**：支持为图片添加标签，便于分类和筛选
- **预设模板**：支持保存和管理提示词模板，快速复用
- **文本处理**：支持文本的替换和追加操作，灵活处理文本内容
- **无缝集成**：输出提示词和文本供其他 ComfyUI 节点使用
- **视觉界面**：提供图像文件的视觉预览，支持卡片视图和列表视图

## 核心功能

### 提示词存储器节点
- **图像文件加载**：从输出目录加载图像文件
- **提示词显示**：显示与图像文件关联的正向和负向提示词
- **提示词选择**：支持多选正向提示词，单选负向提示词
- **视觉预览**：显示图像文件的缩略图，支持左右切换浏览
- **悬停预览**：鼠标悬停显示详细信息预览
- **编辑功能**：支持编辑文件名、标签、风格、提示词等信息
- **标签筛选**：按标签筛选图片
- **标签提示**：多个标签请使用英文逗号`,`分隔，例如：`人像,写实,风景`（请勿使用中文逗号`，`）

### 预设模板存储器节点
- **模板管理**：保存、编辑、删除提示词模板
- **模板分类**：按风格分类管理模板
- **快速应用**：一键应用模板到工作流
- **搜索功能**：快速搜索模板
- **文件格式**：支持读取 TXT、JSON、MD 格式的模板文件

### 文本修改器节点
- **输入接口**：用于接收外部文本内容
- **文本内容**：用于指定被替换的文本内容（作为定位标记），追加时，内容为空则以整段内容信息为准定位
- **模式选择**：支持三种模式
  - 替换(Replace)：用输入接口的内容替换整段文本中与"文本内容"匹配的部分
  - 追加文本前(Append Before)：在"文本内容"位置前插入输入接口的内容
  - 追加文本后(Append After)：在"文本内容"位置后插入输入接口的内容
- **整段文本**：完整的文本内容
- **输出**：处理后的文本内容

## 安装说明

### 1. 基本安装

1. **克隆或下载插件**：
   - 将插件文件夹放入 `ComfyUI/custom_nodes/` 目录
   - 文件夹名称应为 `ComfyUI-prompt-storage`

2. **安装依赖**：
   - 插件除了 ComfyUI 的默认要求外，没有其他额外依赖

### 2. 使用方法

1. **将节点添加到工作流**：
   - 在 ComfyUI 中，从 "prompt" 类别添加所需节点：
     - "提示词存储器(Image Prompt Storage)" - 用于管理图像提示词（缓存目录：`ComfyUI/custom_nodes/ComfyUI-prompt-storage/cache/` 和 `data/`）
     - "预设模板存储器(Prompt Template)" - 用于管理提示词模板（缓存目录：`ComfyUI/custom_nodes/ComfyUI-prompt-storage/templates/`）
     - "提示词选择器(Prompt Selector)" - 用于加载和管理JSON格式提示词文件（缓存目录：`ComfyUI/custom_nodes/ComfyUI-prompt-storage/prompt/`）
     - "文本修改器(Text Modifier)" - 用于文本的替换和追加操作

2. **配置提示词存储器节点**：
   - 设置文件路径为输出目录（存储目录：`ComfyUI/custom_nodes/ComfyUI-prompt-storage/data/`预览图缓存目录：`cache/`）
   - 从下拉列表中选择一个媒体文件
   - 点击"查看详情"按钮打开参数详情界面

3. **配置预设模板存储器节点**：
   - 默认存储目录：`ComfyUI/custom_nodes/ComfyUI-prompt-storage/templates/`
   - 支持 TXT、JSON、MD 格式的模板文件
   - 点击"管理模板"按钮打开模板管理界面
   - 创建、编辑或删除模板

4. **配置提示词选择器节点**：
   - 默认存储目录：`ComfyUI/custom_nodes/ComfyUI-prompt-storage/prompt/`
   - 仅支持 JSON 格式的提示词文件
   - 从下拉列表中选择一个JSON格式的提示词文件
   - 选择输出格式（Text 或 JSON）
   - 点击"新建"按钮创建新卡片或新文件
   - 点击"编辑"按钮编辑选中的卡片
   - 点击"输出选项"自定义输出字段

5. **使用输出**：
   - 节点输出提示词内容
   - 将输出连接到需要提示词输入的其他节点

## 工作流示例

### 基本使用

1. 将 "提示词存储器(Image Prompt Storage)" 节点添加到工作流
2. 从下拉列表中选择一个媒体文件
3. 点击"查看"按钮打开视觉界面
4. 在界面中选择需要的正向/负向提示词
5. 将输出连接到其他节点
6. 执行工作流以使用提示词

### 使用预设模板

1. 将 "预设模板存储器(Prompt Template)" 节点添加到工作流
2. 点击"管理模板"创建或选择模板
3. 将模板输出连接到其他节点
4. 执行工作流

### 使用提示词选择器

1. 将 "提示词选择器(Prompt Selector)" 节点添加到工作流
2. 从文件下拉列表中选择一个JSON提示词文件
3. 点击"新建"按钮创建新的提示词卡片或新文件
4. 在弹出的对话框中填写标题、分类和内容
5. 选择分类或输入新分类名称
6. 点击"保存"完成创建
7. 在卡片列表中选择需要的提示词
8. 将输出连接到其他节点
9. 执行工作流

### 使用文本修改器

1. 将 "文本修改器(Text Modifier)" 节点添加到工作流
2. 将输入接口连接到其他节点的文本输出
3. 在"文本内容"输入框中输入要定位的文本（可选）
4. 选择操作模式：替换、追加文本前或追加文本后
5. 在"整段文本"输入框中输入完整的文本内容
6. 将输出连接到需要文本输入的其他节点
7. 执行工作流
