# ComfyUI-prompt-storage

一个用于管理和存储图片提示词信息的 ComfyUI 插件。

**[📃English](./README.md)**

## 项目简介

ComfyUI-prompt-storage 是一款专门的 ComfyUI 插件，用于管理和存储图片的提示词信息。它允许用户从输出目录加载图像文件，查看其提示词详情，并输出提示词信息以供其他节点使用。

### 核心优势
- **简单直观**：使用方便，界面清晰
- **图像文件支持**：从输出目录加载图片
- **提示词管理**：显示和管理图像文件的提示词信息
- **无缝集成**：输出提示词供其他 ComfyUI 节点使用
- **视觉界面**：提供图像文件的视觉预览

## 核心功能

- **图像文件加载**：从输出目录加载图像文件
- **提示词显示**：显示与图像文件关联的正向和负向提示词
- **提示词输出**：输出选定的提示词供其他节点使用
- **视觉预览**：显示图像文件的缩略图
- **便捷导航**：提供用户友好的界面用于浏览图像文件

## 安装说明

### 1. 基本安装

1. **克隆或下载插件**：
   - 将插件文件夹放入 `ComfyUI/custom_nodes/` 目录
   - 文件夹名称应为 `ComfyUI-prompt-storage`

2. **安装依赖**：
   - 插件除了 ComfyUI 的默认要求外，没有其他额外依赖

### 2. 使用方法

1. **将节点添加到工作流**：
   - 在 ComfyUI 中，从 "prompt" 类别添加 "提示词存储器" 节点

2. **配置节点**：
   - 设置文件路径为输出目录（默认为 ComfyUI 的输出目录）
   - 从下拉列表中选择一个媒体文件

3. **使用输出**：
   - 节点输出正向和负向提示词
   - 将这些输出连接到需要提示词输入的其他节点

## 工作流示例

### 基本使用

1. 将 "提示词存储器" 节点添加到工作流
2. 从下拉列表中选择一个媒体文件
3. 将 "正向提示词" 和 "负向提示词" 输出连接到其他节点
4. 执行工作流以使用提示词
